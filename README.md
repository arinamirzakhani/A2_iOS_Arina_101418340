## Project Description
This iOS application is developed using SwiftUI and Core Data.  
The app manages product information including:

- Product ID
- Product Name
- Product Description
- Product Price
- Product Provider

## Features
- View first product on launch
- Navigate between products (First, Next, Previous, Last)
- Search products by name or description
- Add new products
- View full list of products

## Technologies Used
- SwiftUI
- Core Data
- Xcode
